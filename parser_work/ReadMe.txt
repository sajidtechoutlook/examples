php parser.php --file mycsv.csv  --unique_combination=the_results.csv

Instructions 

1. Console enable
	go to the folder where the script is located and run console in that folder

2. run the following command
	php parser.php --file mycsv.csv  --unique_combination=the_results.csv

3. --file
	place the file in folder where the script is. in command  after --file give space and write the filename with .csv extention that you want to parse

4. --unique_combination 
	is the file name which must created after the parsing is complete. You can give any name with .csv extention.
